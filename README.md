# authentify-page

is a simple authentication system project built with HTML, CSS, and JavaScript. It demonstrates how to create a lightweight login and registration interface with theme switching and responsive design.

Key Features
Authentication Form: Includes username and password fields with validation logic in JavaScript.

Anonymous JSON Database: User data is stored in a basic JSON structure to simulate a database.

Theme Toggle: Supports both light and dark modes with smooth transitions.

Responsive Layout: The interface adapts to different screen sizes, ensuring usability on mobile and desktop.

Separated Files: Code is organized into logical files (index.html, style.css, sys.js) for clarity and maintainability.

## How it work?

- You click on reg button and bowser open new dialogue window, you create account on local DB without saving, and input your account info(Name + password), continue you input login and browser create new dialogue window:
- You logined.
- You not logined.
